# C# финальный проект

## Проект по созданию карт регистрации на C# 
```
Данный проект работает как дополнение к ПК Комита Отчет, програмный комплекс для передачи электронного документооборота.
Может работаь как самостоятельное решение по созданию карт регистрации в формате .xml
```

### Запуск и тестирование проекта
```
1. Скачиваем проект. 
2. Устанавливаем и настраиваем у себя на рабочем месте сервер MSSQL.
3. Устанавливаем базу данных, прикрепленную к проекту.
4. Запускать проект можно из папки: текущий проект\bin\Release запустив файл .exe, также можно открыть проект в Visual Studio и запустить оттуда
5. Можете добавлять новых и редактировать добавленных уполномоченных представителей, все пишется в БД
6. При выборе любого уполномоченного представителя на стартовой странице, все дальнейшие манипуляции будут происходить именно с ним.
7. Загрузите файл "Тестовый файл ПФР" при открытии окна формы "Массовая регистрация"
8. Нажмите кнопку "Поехали"
9. В результате работы будут созданы карты регистрации для указанных в файле абонентов. При желании можно добавить в файле неограниченное число записей новых абонентов, все они будут обработаны.
10. По указанному пути по завершении обработки можно будет посмотреть на полученне файлы
11. Также все абоненты будут добавлены в таблицу в тестовой БД и привязаны к текущему уполномоченному представителю.
